学习使用hugo

安装Hugo

mac安装发现实际上不用clone等一堆的操作，貌似commandtools里就有，如果没有可以执行下brew install hugo 然后，然后就没有然后了，安装完毕

开始玩起来

首先hugo --help大概看了看帮助说明，另外Google了下usage，看下提示，英文不好，这里有中文文档，^O^

下面是我这次使用的一个简单步骤

新建一个site，用以在本地开发

1
2
3
4

```shell
# my-doc 是自己的文件名
hugo new site my-doc
```

选择一个自己喜欢的主题

这里我用的是even, 当然也可以从theme里选择自己喜欢的

操作如下：

# clone 主题到自己工程的themes下
git clone https://github.com/olOwOlo/hugo-theme-even themes/even

# 安装主题
# 进入主题目录
cd themes/even
# 安装，如果这里没有yarn，还有用brew install yarn
yarn install
# build
yarn build
# 回到工程的root目录
cd ...

# 使用主题内的config.toml代替工程初始化的config
cp themes/even/exampleSite/config.toml ./

ok，至此，主题和站点都一初始化好了。下面创建个文档试试。

创建一个文档

新建一个文章hugo new post/new-article.md,然后打开这个md，大致内容如下：

1
2
3
4
5

---
title: "Learn to Use Hugo"
date: 2019-01-20T10:42:40+08:00
draft: true
---

title: 应该就是文章的名字了

date: 自然是时间

draft: 是否为草稿，hugo里文章分草稿和正式稿，草稿如果在build的时候不加上–builddraft是看不到的 其他的嘛，我先慢慢摸索下

本地启动hugo

本地启动比较简单，直接执行hugo server即可，但是我们刚写的是个草稿，所以启动的时候需要加上构建草稿的参数--buildDrafts，不然啥也看不见
